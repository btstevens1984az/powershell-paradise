    B --> C["📊 history.json"]
    B --> D["📝 README"]
    E["⏰ Daily Action"] --> B

|:----:|:-------------|
| 1️⃣ | Query GitHub for `language:powershell` repos with real activity |

| 3️⃣ | Build four bubbly ranking tables — top 15 each |
